Projetos

Naruto BR API

A Naruto BR API é uma API REST desenvolvida em Node.js que disponibiliza dados detalhados sobre personagens, vilas, jutsus e outros elementos do universo de Naruto em português brasileiro (PT-BR). O projeto foi criado para oferecer uma base de dados acessível e de fácil integração, ideal para aplicações educacionais, projetos pessoais ou estudos em desenvolvimento web.

Online Web Tools

Projeto construído utilizando HTML, CSS e JavaScript Puro. São 4 ferramentas, Sorteador de números, Conversor de Texto, Contador de Caracteres e Gerador de Senhas. Este é um projeto pessoal, não utilizei nenhuma biblioteca, o site conta com design responsivo e um botão para alternar entre Dark e Light mode desenvolvido por mim.

Naruto Universe

Naruto Universe é um projeto pessoal que desenvolvi utilizando Vue.js para criar uma interface interativa e visualmente atraente que consome a Naruto BR API. A ideia era trazer os dados da API para a vida, permitindo que os fãs de Naruto explorem informações sobre personagens, vilas, jutsus e muito mais de forma dinâmica e intuitiva.

Consulta Clima e Temperatura

Projeto web usando HTML, CSS e JavaScript para exibir temperatura atual por cidade. Integrada a API do OpenWeather para fornecer dados precisos e em tempo real, entregando uma interface responsiva e informativa aos usuários.

Censo Galáctico

Aplicação que permite visualizar informações detalhadas sobre planetas do universo de Star Wars e seus habitantes famosos. Utilizando a API pública SWAPI, o projeto oferece dados precisos e em tempo real, entregando uma interface responsiva e informativa aos usuários.

PDV Restaurantes Electron + React

(Em Desenvolvimento) - Projeto criado com Electron + React + Sqlite. Se trata de um PDV (Ponto de Venda) para Restaurantes. Este projeto contará com cadastro de items e complementos no cardápio, registro de vendas e relatórios de performance de vendas, entre outras funcionalidades.
